Перевести 0x12345678 в десятичное число

для удобства будем начинать с младшего разряда:
8 * 16^0 + 7 * 16^1 + 6 * 16^2 + 5 * 16^3 + 4 * 16^4 + 3 * 16^5 + 2 * 16^6 + 1 * 16^7  = 305 419 896

0x12345678 =  305419896


Перевести 0x1000000 в десятичное число
для удобства будем начинать с младшего разряда:
0 * 16^0 + 0 * 16^1 + 0 * 16^2 + 0 * 16^3 + 0 * 16^4 + 0 * 16^5 + 1 * 16^6  = 16 777 216

0x1000000 = 16777216
